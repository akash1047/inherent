What Is Inherent?

Inherent is a backend for turning company knowledge into something AI systems can query. Upload documents, let the ingestion pipeline extract and index them, then query via REST or MCP.

Supported Document Formats

  • Plain text (.txt)
  • Markdown (.md)
  • CSV (.csv)
  • HTML (.html)
  • PDF (.pdf)
  • JSON (.json)
  • DOCX (.docx)
